OPPORTUNITIES AT FACEBOSS

At Faceboss, we offer dynamic career opportunities for skilled software developers who want to make an impact. Whether you're a front-end developer, back-end developer, or full-stack engineer, we have positions available for passionate tech enthusiasts who thrive in a collaborative and fast-paced environment.

Current Openings
Software Developer

We are looking for skilled software developers who are passionate about building robust and scalable applications. Ideal candidates should have experience in modern programming languages and frameworks.

Web Developer

As a web developer at Faceboss, you will work with cutting-edge technologies such as HTML, CSS, JavaScript, and popular web frameworks like React, Vue, or Angular. The ideal candidate will have a deep understanding of web design and responsive development.

Mobile App Developer

We are seeking a mobile app developer with experience in Android, iOS, or cross-platform development using frameworks like Flutter or React Native. The ideal candidate will have a passion for creating seamless mobile experiences.

Software Tester

Our software testers are responsible for ensuring our applications run smoothly and meet the highest quality standards. Experience with automated testing tools and a keen eye for detail is essential.

Database Administrator

We need a skilled database administrator who can manage and optimize our databases. Experience with SQL, NoSQL, and cloud-based database management systems like AWS RDS or Azure SQL is required.

Technical Support

As part of our technical support team, you will assist users with technical issues and ensure our systems are running smoothly. Ideal candidates should have excellent troubleshooting skills and a customer-first mentality.
